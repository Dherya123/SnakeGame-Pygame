# 🐍 Snake Game using Python & Pygame

A classic Snake Game developed using Python and Pygame. Control the snake, eat food to increase your score, and survive as long as possible without hitting the walls or your own body.

## 🎮 Features

- Smooth snake movement
- Food spawning system
- Collision detection
- Simple and clean user interface

## 🛠️ Technologies Used

- Python 3
- Pygame
